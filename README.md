# C² Community

This repository is the source repository for the public GitHub Discussions of
the C² project.

C² is built around three connected resources:

1. YouTube videos for guided explanations.
2. Free interactive laboratories for experimentation.
3. Public GitHub repositories for code, circuits, tests, and verification.

## Community

Use the [organization Discussions page](https://github.com/orgs/c2-foundations/discussions)
for conceptual questions, circuit help, unexpected behaviour, and suggestions.

The community is shared by:

- **Cientos de Cimientos** — Spanish.
- **Countless Concepts** — English.

Choose the category for your language and the laboratory you are using.

## Support and security

Read [SUPPORT.md](SUPPORT.md) before opening a technical request.

Do not disclose vulnerabilities in public Discussions. Follow
[SECURITY.md](SECURITY.md) and the private reporting flow of the affected
repository.

## Project principles

The essential educational content and laboratories are intended to remain
complete, free to use, and publicly inspectable.
